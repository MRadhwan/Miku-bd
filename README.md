# 🏛️ THE SILENT ARCHITECT | Project: Miku-bd (v500)

## 📄 PROJECT OVERVIEW
**A bespoke, interactive digital experience created for Miku's 17th birthday.** This application was designed to move beyond the limitations of a standard digital card, offering a multi-layered journey built on custom logic, security verification, and interactive mini-games.

## 🛠️ CORE ARCHITECTURE & FEATURES

### 🔐 Security & Entry
* **Passcode Gateway:** Access is restricted via a **** verification code.
* **Interactive Hints:** A custom-coded hint mechanic requires physical interaction to reveal access clues.

### 🎭 Dynamic Personalization
* **Grandma Mode Toggle:** A state-specific filter that applies a sepia/grayscale overlay to the entire DOM.
* **Contextual Audio:** Dual-track playback that adapts based on user selection.

### 🎮 Interactive Mini-Games
* **The Cake Unboxing:** A multi-stage interactive sequence built with CSS animations.
* **The Football Finale:** A physics-based striker challenge that serves as the application's exit gate.

### ⚡ Technical Performance
* **Offline Ready:** Service Worker integration ensures the experience is cached and available without an internet connection.
* **Mobile-First Design:** Optimized specifically for performance on the Samsung S24 FE.
